# Role-Based Access Control Summary

## Role Permissions Matrix

### Student Role
- **Dashboard** (Read only)
- **Homework** (Read only)
- **Reports** (Read only)
- **Results** (Read only)
- **Timetable** (Read only)

### Parent Role
- **Dashboard** (Read only)
- **Homework** (Read only)
- **Reports** (Read only)
- **Results** (Read only)
- **Timetable** (Read only)

### Subject Teacher Role
- **Dashboard** (Read/Write)
- **Homework** (Read/Write)
- **Reports** (Read/Write)
- **Results** (Read/Write)
- **Timetable** (Read/Write)

### Class Teacher Role
- **Dashboard** (Read/Write)
- **Attendance** (Read/Write)
- **Fees** (Read only)
- **Homework** (Read/Write)
- **Reports** (Read only)
- **Results** (Read/Write)
- **Timetable** (Read/Write)

### Other Staff Role
- **Dashboard** (Read only)

### Accountant Role
- **Dashboard** (Read only)
- **Attendance** (Read only)
- **Fees** (Read/Write)
- **Payroll** (Read/Write)

### Principal Role
- **All Modules** (Full access)
- Cannot access Users module

### Admin Role
- **All Modules** (Full access)
- **Users** (Full access)

### Super Admin Role
- **All Modules** (Full access)
- **Users** (Full access)
- **Protected from deletion/role changes**

## Permission Levels
- **Read**: Can view data
- **Write**: Can create and modify data
- **Delete**: Can remove data
